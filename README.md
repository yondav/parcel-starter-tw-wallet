# parcel-starter-tw-wallet

## Getting Started

```

npx @ydstarters/parcel-starter-tw-wallet <APP_NAME>


```

rename `.env.EXAMPLE` to `.env`

```

REACT_APP_INFURA_ID=<YOUR INFURA ID>

```

```

cd <APP_NAME> && yarn start

```

Have fun...
